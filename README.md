# PRICE COMPARISON WEBSITE
## Installing requirements
```bash
pip install -r requirements.txt
```

### Running the project

1. Clone the repository
2. Create a virtual environment and activate it
3. Install the requirements
4. ```bash 
4. python manage.py runserver
4.  ```
